经典ICG博弈算法(Impartial Combinatorial Games 公平组合游戏)

  1. 有两名选手;
  2. 两名选手交替操作，每次一步，每步都是在有限的合法集合中选取一种进行;
  3. 在任何情况下，合法操作只取决于情况本身，与选手无关;
  4. 游戏的败北条件为:当某位选手需要进行操作时，当前没有任何可以执行的合法操作，则该选手败北。

巴什博弈 Bash Game

题目：一堆n个物品，两个人轮流从中取出1~m个，最后取光者胜（不能继续取的人输）。

同余定理：n=k∗(m+1)+r，先者拿走r个，那么后者无论拿走1 m个先者只要的数目使和为m+1，那么先手必赢。反之若n=k∗(m+1)，那么先手无论怎样都会输。

三个性质

无后效性

未来状态只关心过去状态的结果，不关心过去状态的过程。

最优子结构

最优子结构性质：一个问题的最优解由其子问题的最优解推出。

  1. 证明问题最优解的第一个组成部分是做出一个选择；
  2. 对于一个给定问题，在其可能的第一步选择中，假定你已经知道哪种选择才会得到最优解。你现在并不关心这种选择具体是如何得到的，只是假定已经知道了这种选择；
  3. 给定可获得的最优解的选择后，确定这次选择会产生哪些子问题，以及如何最好地刻画子问题空间；
  4. 证明作为构成原问题最优解的组成部分，每个子问题的解就是它本身的最优解。方法是反证法，考虑加入某个子问题的解不是其自身的最优解，那么就可以从原问题的解中用该子问题的最优解替换掉当前的非最优解，从而得到原问题的一个更优的解，从而与原问题最优解的假设矛盾。

问：你怎么看待第十名现象？ (中学里成绩徘徊在第十名左右的学生长大后的出息有时比第一名大)

答：这个名词说实在的我还是第一次听到， 也不知道这一现象有多大的普遍性。 不过以我个人的经历为背景， 我想这其中的道理可能是 “第十名” 的学生往往比 “第一名” 的学生在个人兴趣的发展上更为超前， 更为投入， 或者说在学习上更具个性。 中学的功课文理兼备， 涵盖的学科十分广泛， 广泛到了很少有人会对所有这些功课同时感兴趣的程度。 应该说无论是 “第一名” 还是 “第十名” 都属于名列前茅， 这样的学生通常都具有较强的学习能力。 但是为了获得总成绩的第一名， 需要学生在所有功课上都投入相当的精力， 这就往往会形成一个杂而不精的局面。 相反， 一个 “第十名” 的学生往往能够在学习上有所侧重 (有所侧重当然也就意味着有所偏科， 因此只能得 “第十名”)， 对自己喜爱的学科能够超越课本、 超越课堂， 达到普通同学 (包括第一名的同学) 难以达到的深度。

另一方面， 随着知识一代一代地持续积累， 通才的存在变得越来越不可能， 社会的发展在越来越大的程度上有赖于分工， 有赖于学有专长的个体之间的互补。 在这种大环境下， 一个学有专长的 “第十名” 学生往往会比一个博学强记的 “第一名” 学生更能胜任社会所赋予的角色。 我想这就是第十名现象出现的主要原因。

观点自带立场，事实越辩越清

发表于 2023-02-25 更新于 2023-05-01 分类于 随笔 本文字数： 675 阅读时长 ≈ 2 分钟

尽管新闻报道很强调客观性，但是在实际中，新闻报道或多或少都会受到个人的喜好憎恶和阶层立场、时代背景的影响，所以文章的观点必然是有立场的，只看某一篇报道或者某一种倾向的报道（比如日本右翼对二战的报道）往往不能了解事情的全貌，也不能通过阅读收获什么有价值的东西，不能对事情的发生进行合理的解释。

读新闻要区分新闻的真假，并不是所有的新闻都是在“帮”你。

现在假设给你一些突发新闻的报道，你要对其进行整合，你会如何进行筛选精简，提炼观点？

很显然，不同人根据自己的想法，对某些方面进行侧重保报道，忽略或者掩盖某些方面的报道，以表达个人的某种倾向和态度。

事件的看法会受到时代的局限性的影响。例如一些农民起义的失败，用封建王朝的理论去解释和用历史唯物主义去解释，所得的结论也是不同的。

事件的看法会受到阶层立场的影响，例如白左和普通群众对特拉斯金融政策的态度。

一些人一直在强调新闻要绝对客观，这种观点本身就是不现实的，没有客观分析过的。

我有一个忘记从哪继承来的观点：无论我们想学什么，都应该从学习他的历史开始。极端点说，无论学什么，都是在学他的历史。

前几天在北京见到了我的小表妹，她刚上初一，人极为聪明。七年前我在北京工作时，她还没有上小学。那个时候我就回答过她许多科普问题。一个五岁的小姑娘的理解能力曾让我感叹不已。

知道小妹妹居然在写 blog 而且文笔不错，觉得挺高兴的。但听说她最近一次考试数学成绩不是特别好，有些令人担忧。我一直认为中学所谓文理分科就是件巨傻的事（似乎现在慢慢的不再分了）。但如果一味的去追求所谓文科上的造诣，怕是会耽误理性思维的发展。在迈入科学的殿堂的门槛上，一步错就能耽误十年啊。

小孩子一开始在学校学习或许更多的是为了比同龄人强，为了完成长辈的期望，为了达到别人设定的目标。但终有一天，长大了的后会发现，学习其实就是因为自己想知道。

所以吃饭的时候，我多说了几句。无论学什么，都要先培养出兴趣。理科的东西老师教起来很枯燥，那么就可以从读他们的历史开始。一段段故事，一个个鲜活的人，就会变得生动。深一层的道理需要慢慢的去体会：无论学什么，都是在学他的历史。以前的人如何考虑这些问题，为何研究这些问题。他们怎样去归纳、总结。他们的思考怎样延伸出去，为什么又有了局限性。后人怎样做出了突破。人性是共通的，人的智力水平也相差无多。按着前人的路，没有太难理解的道理。而课堂上现在的教法，把历史上长长的思考过程压缩，裁减掉所有的错误和累赘，压缩成一条条公式与冷冰冰的推导。背了那些，除了考出完美的答卷，就没太多意义了。

数学归纳法的等价定义

基本条件：

  1. 当 \(n=1\) 时，\(T\) 成立；
  2. 对任意 \(n>1\)，如果 \(n-1\) 时 \(T\) 成立，那么 \(n\) 时 \(T\) 也成立

定义：如果对于带有参数 \(n\) 的命题 \(P\) ，当 \(n=1\) 时 \(Р\) 成立；并且如果对每一个 \(n (n>1)\),若 \(n-1\) 时 \(P\) 成立能推出 \(n\) 时 \(Р\) 也成立；那么对任意自然数，\(Р\) 都成立。

等价一：如果对于带有参数n的命题P，当n= 1时Р成立;并且如果对每一个n (n ≥ 1 ),若n时Р成立能推出n+1时Р也成立;那么对任意自然数，Р都成立。

等价二：如果对于带有参数n的命题P，当n=1时Р成立;并且如果对每一个n (n> 1 ),若对任意小于n的自然数Р成立能推出对n命题Р也成立;那么对任意自然数,Р都成立。

等价三：如果对于带有参数n的命题P，当n=1和n=2时Р都成立;并且如果对每一个n (n >2)，若n-2时Р成立能推出n时Р也成立;那么对任意自然数，Р都成立。

等价四：如果对于带有参数n的命题P，当n=1和n=2时Р都成立;并且如果对每一个n (n >2)，若n-2时Р成立能推出n时Р也成立;那么对任意自然数，Р都成立。

关于竞争

发表于 2023-02-10 更新于 2023-04-24 分类于 随笔 本文字数： 434 阅读时长 ≈ 2 分钟

竞争是人本能的一种思维习惯，人类从诞生之日起就在把自己和别人进行比较，偶尔又在把自己和自己进行比较。

和别人竞争不一定是坏事，竞争能激发人潜在的动力，激起人内心的兽性；但是竞争的结果未必总是好的，资源之所以要被争夺，被选拔，本质上就是因为资源是有限的，一部分人能享受到，意味着另一部分人就不能享受到。

人如果长时间投入竞争的关系当中，就容易陷入结果主义当中，过分得相信自己有了那么大的投入就一定能有收获，然而这只能是增加了获胜的概率而已，如果人过分得看重竞争的结果，一旦面对竞争失败的结果时，就很容易自暴自弃，放弃继续竞争的动力。

对于竞争本身，我们既不应该只享受结果本身，应该看到我们在奋斗的过程中也是值得享受的，衡量每天的进步的标准应该是我比我的过去进步了多少，而不是我和别人相比我如何如何，只要个体在进步，尽管竞争失败也是一种成功。

当然这并不意味着我们可以一直失败，人在失败的状态下想要进步的成本会变得很高，想要摆脱逆境需要我们继续付出时间，在这个状态下，需要我们付出更多的精力，如果目标定的不合适也需要适当得调整目标（比现实高一步）。
